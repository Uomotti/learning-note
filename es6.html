# ES6

  • ECMA Script 2015 와 동일한 용어
  • 2015년은 ES5(2009년) 이래로 진행한 첫 메이저 업데이트 된 해
  • 문법이 간결해져서 코딩을 더 편하게

# const & let

let a = 10;
let b = 10;

let으로 재 정의시 에러 발생.

const a = 10;

const 로 정의시 재정의, 변수 값 변경 완료

# Enhanced Object Literal (향상된 객체 리터럴)

var num = 10;
var obj = {
  // num : num 와 같음
  num,
};

# arrow function

function arrowFunc(a, b) {
  return a + b;
}

// es6 문법

arrowFunc = (a, b) => {
  return a + b;
};
