# vuepress

  1. Node.js 설치


2. https://vuepress.vuejs.org/ --> Getting Started 
npm install -g vuepress 
npm install -D vuepress


  1. package.json 아래 내용 추가
{
  "scripts": {
    "doc": "vuepress dev docs",
    "build": "vuepress build docs"
  }
}


  1. node 명령어실행
npm run doc
